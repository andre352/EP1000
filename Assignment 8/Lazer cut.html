Lazer Cutting

On this page, I will be showing on how I used Fusion 360 to create a box and lazer cut the box which is made out of plywood.





Lazer cut a box

Step 1

The first step is to open Fusion 360 and set the parameters for the box so that it is easier to create the box.





Step 2

The second step is to create the box using the parameters set up in the step above.





Step 3

The third step is to click on one of the sides (turn blue) and create sketch. Then left click the side again so that it is highlighted and then finish sketch. Do that for all the sides.





Step 4

In this step, we save the file you created in the previous step as a DXF file. This wil be needed to lazer cut the box





Step 5

The next step is to open LibreCAD and import block (the DXF files). This step will fit all the parts into a DXF file for it to be lazer cut.





Step 6

The last step is to send the DXF file to lazer cut and assemble the parts that are lazer cutted. Enjoy the product that is created.
